# Support

## How to Get Help

Use GitHub Issues for:
- Bugs
- Feature requests
- Documentation problems

Use GitHub Discussions (if enabled) for:
- Questions
- Usage advice

## Before Opening an Issue
1. Read `README.md`.
2. Read user docs in `docs/en` or `docs/ru`.
3. Check existing open/closed issues.
4. Include logs, steps to reproduce, and expected vs actual behavior.
