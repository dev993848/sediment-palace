# Code of Conduct

## Our Pledge
We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Being respectful and constructive.
- Accepting feedback gracefully.
- Focusing on what is best for the community and the project.

Examples of unacceptable behavior:
- Harassment, insults, or personal attacks.
- Trolling, deliberate intimidation, or discrimination.
- Publishing private information without permission.

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing this code of conduct.

## Scope
This Code of Conduct applies within project spaces and in public spaces when representing the project.

## Reporting
Report issues to maintainers via repository discussions/issues with the `conduct` label.

## Enforcement
Maintainers may remove, edit, or reject comments, commits, code, and issues that violate this Code of Conduct.
